# 🔐 Login System Project

## 📌 What this project does

This is a simple login system where a user can:

* Create an account (signup)
* Login using email and password
* Keep their password secure (not stored in plain text)
* Get a token after login

---

## 🧠 How it works

* When a user signs up:

  * The password is converted into a secure format (hashed)
  * Then it is stored in the database

* When a user logs in:

  * The system checks if the password is correct
  * If correct → login is successful and a token is given

---

## 🎯 Why this project

* To understand how login systems work
* To learn basic backend development
* To practice authentication using tokens

---
